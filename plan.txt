APP DE CONTROL DE GASTOS PERSONALES
Especificación del sistema

1. OBJETIVO
   Crear una aplicación de escritorio para registrar, analizar y controlar los gastos personales mensuales.
   La aplicación permitirá almacenar gastos, analizarlos mediante estadísticas y ayudar al usuario a controlar su presupuesto.

---

2. TECNOLOGÍAS RECOMENDADAS

Lenguaje:
Python

Interfaz gráfica:

Base de datos:
SQLite

Librerías adicionales:
matplotlib

datetime
json

openpyxl
